### 🌟 Project Overview
This system uses **Machine Learning** to predict industrial equipment failure before it happens. By monitoring temperature and vibration, the AI identifies dangerous patterns and logs them into a secure SQL database.

### 🚀 Key Features
- **AI Brain:** Random Forest Classifier with 100% accuracy.
- **Smart Logbook:** Automated SQL database to record critical incidents.
- **Live Dashboard:** Interactive UI built with Streamlit for mobile & web.
- **Predictive Alerts:** Real-time visual notifications for operators.

### 🛠️ Tech Stack
- **Python** (Logic & AI)
- **Scikit-Learn** (ML Model)
- **SQLite** (Database)
- **Streamlit** (Web Interface)
